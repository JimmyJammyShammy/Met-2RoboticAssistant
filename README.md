# Met-2RoboticAssistant - Started 02.02.2026
This repository contains code for the Met^2 Robotic Assistant Project, including functionality for:

  - Sending instruction to stepper motors via ESP32 pins through a TB6600 Stepper Motor Driver
    - Left, Right, Arm, Fridge Stepper Motors
  - Recording sensor data and translating into movement
    - Color Sensor input (TCS34725 Color Sensor)
    - Ultra-Sonic Sensor input
  - Movement Instruction Recording
    - Save and replay movement sequences for repeated action
  - Analog Selection System
    - Communicates with the food container system
